# GD_caption
A faster way to manually caption images

Setup:
 - Setup WhisperAI
 - Run python script

Features:
- Loads Image Folder
- Loads Associated .txt file
- Remove Image from directory



Shortcuts:

- [Left] : Previous Image
- [Right] : Next Image
- [Enter] : Saves Caption and Proceeds to Next Image
- [L-shift] : record audio (Hold while recording)

Still in Development.
